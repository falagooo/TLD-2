# Простой сайт «Мои соцсети»

Файлы находятся в этой папке. Описание:

- `index.html` — главная страница
- `styles.css` — стили
- `script.js` — логика сохранения в localStorage

Как запустить локально:

1. Откройте `index.html` в браузере (двойной клик или "Открыть в браузере").
2. Заполните поля с ссылками и нажмите "Сохранить". Данные сохраняются в localStorage вашего браузера.

Особенности:
- 4 блока для произвольных соцсетей.
- Отдельный блок под Telegram (под ним отдельная кнопка-превью).
- Кнопка "Очистить" удаляет сохранённые ссылки.

Фоновое видео:
- Положите файл видео в папку `video` внутри этого проекта и назовите его `11_2_stab_prob4.mp4` (путь: `video/11_2_stab_prob4.mp4`).
- Видеофайл будет автоматически воспроизводиться, зациклен и выключен звук (muted). Если браузер блокирует автозапуск, попробуйте открыть через локальный HTTP-сервер.
 - Файл будет пытаться проигрываться со звуком автоматически. Однако многие браузеры по умолчанию блокируют автозапуск видео со звуком.
 - Если автозапуск со звуком не происходит, возможные обходы:
	 - Откройте сайт через локальный HTTP-сервер (иногда это помогает).
	 - Взаимодействуйте с вкладкой (клик или клавиатура) — браузер после этого разрешит звук.
	 - Измените настройки автопроигрывания в браузере (не рекомендуется для обычных пользователей).

Обновление: стеклянный (glassmorphism) стиль
------------------------------------------------

Я добавил современный "стеклянный" прозрачный стиль к колонкам, кнопкам и окнам:

- Новые CSS-классы: `glass`, `glass-window`, `glass-button`, `notice`.
- Применено к карточкам соцсетей, блоку Telegram и к основным кнопкам.
- Наличие fallback для браузеров без `backdrop-filter` и поддержка `prefers-reduced-motion`.

Как посмотреть изменения локально:

1. Откройте `index.html` прямо в браузере — большинство изменений будут видны, но на некоторых браузерах `backdrop-filter` может не работать при открытии файла напрямую.
2. Рекомендуемый способ: запустить простой локальный HTTP-сервер в папке проекта. Например, с PowerShell (Windows) выполните:

```powershell
# Откройте PowerShell в папке проекта, затем:
python -m http.server 8000
# или, если у вас Python 2:
python -m SimpleHTTPServer 8000
```

Затем откройте в браузере: http://localhost:8000

Короткие заметки по доступности:
- Кнопки-превью получают `aria-disabled` и визуально приглушаются, если ссылка не задана.
- При снижении движений (`prefers-reduced-motion`) эффекты размытия и тени уменьшаются.

Если хотите, могу настроить более тёмную или более яркую версию стекла, добавить градиенты, или включить переключатель темы (light/dark).
